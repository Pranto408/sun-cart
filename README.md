# 🌞 SunCart

A modern summer eCommerce platform where users can explore and purchase seasonal products like sunglasses, summer outfits, skincare, beach accessories, and more. Users can browse products, view details, and place orders after authentication.

---

## 🌐 Live URL

🔗 [https://sun-cart-two.vercel.app](https://sun-cart-two.vercel.app)

---

## ✨ Key Features

- 🛍️ **Product Catalog** — Browse seasonal products across categories: sunglasses, outfits, skincare, and beach accessories
- 🔐 **Authentication** — Secure sign-up, login, and session management powered by Better Auth
- 📦 **Order Placement** — Authenticated users can place and track their orders
- 🎨 **Modern UI** — Responsive design with smooth animations and a summer-inspired aesthetic
- 🔔 **Toast Notifications** — Real-time feedback for user actions like login, errors, and cart updates

---

## 📦 npm Packages Used

| Package | Purpose |
|---|---|
| [Next.js](https://nextjs.org/) | React framework for routing, SSR, and API routes |
| [Better Auth](https://www.better-auth.com/) | Authentication — sign up, login, and session handling |
| [MongoDB](https://www.mongodb.com/) | Database for storing user data and orders |
| [Tailwind CSS](https://tailwindcss.com/) | Utility-first CSS framework for styling |
| [HeroUI](https://www.heroui.com/) | Pre-built accessible UI component library |
| [Lucide React](https://lucide.dev/) | Clean, consistent icon set |
| [React Icons](https://react-icons.github.io/react-icons/) | Extended icon library |
| [Animate.css](https://animate.style/) | Ready-to-use CSS animation classes |
| [React Toastify](https://fkhadra.github.io/react-toastify/) | Toast notification system for user feedback |
